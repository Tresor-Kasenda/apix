# apix Logo

Files:
- `apix-mark.svg`: square icon mark for avatars and app icons.
- `apix-logo.svg`: full logo for light backgrounds.
- `apix-logo-dark.svg`: full logo for dark backgrounds.

Recommended usage:
- Header/README: `apix-logo.svg`
- Dark UI: `apix-logo-dark.svg`
- Favicons/social avatars: `apix-mark.svg`
